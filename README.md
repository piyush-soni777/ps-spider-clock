# 🕷️ Spider Clock

Spider Clock is a creative and visually stunning analog clock built using HTML, CSS, and JavaScript.  
This project features a unique spider-themed design where the clock hands are creatively integrated with a spider and web animation.

---

## 🎯 Live Demo
🔗 https://ps-spider-clock.netlify.app

---

## ✨ Features
- 🕷️ Unique Spider-themed clock design  
- 🕒 Real-time analog clock functionality  
- 🎨 Smooth animations and visual effects  
- ⚡ Fully powered by JavaScript  
- 📱 Responsive design  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ How It Works
- JavaScript fetches the current time  
- Clock hands rotate based on hours, minutes, and seconds  
- CSS animations create a smooth and aesthetic visual experience
